# لماذا صنعت StayZen؟ - قصة تطبيق لمنع الدخول في وضع السكون Mac

## المقدمة

مرحباً! أنا دحمان مطور تطبيقات ويب ، وفي هذه التدوينة سأشارككم قصة إنشاء StayZen ولماذا قررنا بناءه.

## المشكلة

هل واجهت يوماً هذه المواقف؟

- تحمّل ملف كبير وتحتاج للمغادرة
- تشاهد فيلم طويل وتحتاج للذهاب للحمام

لكن بمجرد ترك الكمبيوتر، يدخل في وضع السكون وتوقف كل شيء!

## الحل الأول: caffeinate

منذ سنوات، كنت أستخدم أمر `caffeinate` في Terminal لمنع نوم Mac:

```bash
caffeinate -i -s &
```

لكن هذا الحل كان:

- صعب الاستخدام للمبتدئين
- لا يوفر واجهة رسومية
- لا يمكن تتبع الوقت أو معرفة حالة الجهاز

لذلك قررت بناء تطبيق يوفر:

### 1. سهولة الاستخدام

بدلاً من كتابة أوامر،, نقرات بسيطة تخليك تقدر تسوي المطلوب وواجهة بسيطة

### 2. وضعان للتشغيل

- **وضع الوقت**: حدد المدة التي تريدها (30 دقيقة، ساعة، ساعتين، أو وقت مخصص)
- **وضع المراقبة**: ابقِ Mac مستيقظاً أثناء تشغيل تطبيق معين

### 3. المتابعة

- سجل للجلسات السابقة
- الوقت الإجمالي
- إشعارات عند بدء/إيقاف منع النوم

### 4. التكامل مع النظام

- يعمل في شريط القوائم
- تشغيل تلقائي عند الدخول
- واجهة رسومية جميلة

## التقنيات المستخدمة

- **الواجهة**: React + TypeScript + Vite
- **الباك اند**: Go
- **سطح المكتب**: Electron
- **النظام**: macOS caffeinate, pgrep

## لماذا مجاني؟

التطبيق مجاني تماماً بدون أي رسوم. لماذا؟ لأن:

1. **بناء مشروع تعليمي**: تعلمت الكثير أثناء تطويره
2. **مساهمة للمجتمع**: أريد مساعدة المطورين والأشخاص الذين يحتاجون مثل هذا الأداة
